# Codex Pets Viewer

Portable Windows viewer for bundled Codex-style pet animations.

## Included pets

- Little Black Mage
- Merry
- Mimi
- Veiglet

## Usage

- Left-click drag: move the pet
- Right-click: open the pet menu
- Mouse wheel or `+` / `-`: resize
- Hover: show controls
- `Esc`: quit

## Build

```powershell
npm install
npm run build:win
```

The portable EXE is written to `dist/Codex Pets Viewer.exe`.
